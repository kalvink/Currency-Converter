# CurrencyConverter
Basic Currency Converter coded in Java by Kalvin Kao

Functionality
- Addition
- Subtraction
- Multiplication
- Division
- Equate
- Decimals
- Clear all
- Back space

Tools used
- Eclipse
- Eclipse WindowBuilder plugin

APIs used
- https://exchangeratesapi.io/

Libraries
- Google GSON
